# 湖北农业农村重大项目监测管理平台

## 项目简介
本项目是一个全栈 Web 应用程序，用于监测和管理湖北省“十五五”农业农村重大项目。

## 永久性开发规则
1. **地图引擎**：禁止使用 Mapbox。后续所有地图单元必须使用 **天地图**。
2. **天地图 Token**：`b5de8c428b2dbb5e9894f02ffdc5cff8`。
3. **技术栈**：Vue 3 + Vite + Tailwind CSS + MapLibre GL JS + ECharts。

## 运行与开发
- `npm run dev`：启动开发服务器。
- `npm run build`：构建生产版本。
